ガイド

ヨコサワのトーナメント用ハンドランキング（8段階の色分けシステム）を覚えるための練習アプリです。

ポーカーのプリフロップ（最初のベッティングラウンド）で、どのハンドをプレイし、どのハンドを降りるかを素早く判断できるようになることを目指します。

4つのシナリオ

  • ミックス — オープンとvsオープンがランダムに出題されます。
  • オープン — 自分が最初にレイズする場面。ポジションに応じたレンジでRAISE or FOLD。
  • vs オープン — 前のプレイヤーがレイズした場面。3BET / CALL / FOLD を判断。
  • 復習 — 過去に間違えた問題だけを出題。苦手なハンドを集中的に練習できます。

オープンでRAISEが正解だった場合、40%の確率で後ろのプレイヤーが3betしてくる「vs 3bet」問題にチェーンします。

クイズ画面

テーブル上にポジションとハンドが表示されます。状況に応じて正しいアクションを選んでください。

  • シナリオとポジションのフィルターで出題範囲を絞れます
  • 正解数・正答率・連続正解数が上部に表示されます
  • オープンでRAISEした後、40%の確率で3bet問題に発展します

キーボードショートカット

  • R RAISE / 3BET / 4BET（攻撃アクション共通）
  • C CALL
  • F FOLD
  • クイズ画面下部のキーバッジをタップすると、キーを自由に変更できます

レンジ表

13×13のグリッドで各ハンドのティアを色分け表示します。ポジション別にフィルターすると、そのポジションでプレイするハンドだけがハイライトされます。

履歴

過去の回答を一覧で確認できます。正解/不正解でフィルターして、苦手なハンドを復習しましょう。

ティア（色）の定義

ハンドは強さ順にT1〜T8の8段階に分類されます。
番号が小さいほど強いハンドです。

ティア色意味主なハンド例
T1紺プレミアAA, KK, QQ, AKs, AKo
T2赤8人中JJ, TT, 99, AQs, AJs
T3黄8人弱88, 77, KJs, QJs, JTs
T4緑6〜7人66, 55, A9s-A2s, KTs
T5水色4〜5人44-22, Q9s, J9s, 98s
T6白3人K8s-K2s, 87s, 76s
T7紫2人Q5s-Q2s, 86s, 54s
T8ピンクBB対BTN専用J5s-J2s, T6s, 85s

ポジションとオープンレンジ

9人テーブル。後ろの人数がそのまま「何色まで参加できるか」を決めます。

ポジション後ろ人数参加できる範囲
UTG8人 T1〜T3（黄まで）
UTG+17人 T1〜T4（緑まで）
MP6人 T1〜T4（緑まで）
LJ5人 T1〜T5（水色まで）
HJ4人 T1〜T5（水色まで）
CO3人 T1〜T6（白まで）
BTN2人 T1〜T7（紫まで）
SB1人 T1〜T7（紫まで）
BB0人オープン不可（既にBBを出している）

オープン（自分が最初にレイズ）

シンプルに「そのポジションの範囲内かどうか」だけ見ます。

ハンドの色がポジションの範囲内 → RAISE 範囲外 → FOLD

例: HJでAJo → AJoは黄(T3)、HJは水色(T5)まで → 範囲内なので RAISE
例: UTGで66 → 66は緑(T4)、UTGは黄(T3)まで → 範囲外なので FOLD

vs オープン（前の人のレイズに対して）

相手のオープン範囲（色）を基準に、自分のハンドがどれだけ上かで判断します。

2色以上強い → 3BET（自信を持ってリレイズ） 1色だけ強い → CALL（コールで参加） 同色かそれより弱い → FOLD

「色の差」とは、相手のオープン範囲の上限ティアと自分のハンドのティアの差です。

例: COオープン → COの範囲は白(T6)まで
　BTNでT9s緑(T4) → 白から2色上 → 3BET
　BTNでJ9s水色(T5) → 白から1色上 → CALL
　BTNでK8s白(T6) → 同色 → FOLD

BBの特殊ルール（vs オープン）

BBは既に1BBを出しているのでお得にコールできます。
通常よりCALL範囲が広がります。

オープナーCALL範囲わかりやすく言うと
BTNT8（ピンク）までほぼ全部コール
COT7（紫）までかなり広くコール
それ以外T6（白）まで防衛的にコール

※ 3BETの条件（2色以上強い）は通常と同じです。CALL範囲だけ広がります。

vs 3bet（3betされた後の対応）

自分のオープン範囲を基準に、ハンドがどれだけ上かで判断します。

T1（プレミア）→ 常に4BET（絶対降りない） 4色以上強い → 4BET 2〜3色強い → CALL 1色以下 → FOLD

例: BTNオープン → BTNの範囲は紫(T7)まで
　KJs黄(T3) → 紫から4色上 → 4BET
　A9s緑(T4) → 紫から3色上 → CALL
　J9s水色(T5) → 紫から2色上 → CALL
　K8s白(T6) → 紫から1色上 → FOLD
