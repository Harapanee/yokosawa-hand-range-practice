# Duolingo風ポップUI リデザイン設計書

## 概要
ヨコサワハンドレンジ練習アプリのUIを、現在のカジノ風ダークテーマからDuolingo風のポップで明るいデザインに全面変更する。

## ターゲット
- スマホ中心利用
- ポップ・ビビッドなテイスト

## カラーパレット

| 用途 | カラー | 説明 |
|------|--------|------|
| 背景 | `#f7f7f7` | ライトグレー |
| カード背景 | `#ffffff` | 白 |
| RAISE | `#58cc02` | Duolingo風グリーン |
| RAISE底面 | `#46a302` | ボタン3D効果用 |
| FOLD | `#ff4b4b` | ビビッドレッド |
| FOLD底面 | `#cc3c3c` | ボタン3D効果用 |
| 3BET | `#ce82ff` | パープル |
| 3BET底面 | `#a568cc` | ボタン3D効果用 |
| CALL | `#1cb0f6` | スカイブルー |
| CALL底面 | `#1899d6` | ボタン3D効果用 |
| テキスト | `#3c3c3c` | ダークグレー |
| サブテキスト | `#afafaf` | ミディアムグレー |
| ストリーク強調 | `#ff9600` | オレンジ |
| 正解フィードバック | `#58cc02` | グリーン |
| 不正解フィードバック | `#ff4b4b` | レッド |

## フォント
- メイン: `Nunito` (Google Fonts) - 太め丸ゴシック系サンセリフ
- 数値/モノスペース: `JetBrains Mono` (継続使用)

## コンポーネント設計

### ボタン (RAISE/FOLD/3BET/CALL)
- Duolingo特有の3Dボタン: `border-bottom: 4px solid [darker-color]`
- `:active`で`border-bottom`を縮めて沈む効果
- 角丸: `16px`
- フォント: `Nunito`, 700, 18px, letter-spacing: 2px
- 白テキスト + 軽いtext-shadow

### ポーカーテーブル
- フェルトテクスチャ廃止
- シンプルな楕円: 薄いグリーン背景(`#e8f5e9`) + 1px ボーダー(`#c8e6c9`)
- ウッドリム廃止 → 軽いbox-shadow のみ

### 座席バッジ
- 丸型バッジ継続
- デフォルト: 白背景 + ライトグレーボーダー + ダークグレーテキスト
- ヒーロー: `#58cc02`背景 + 白テキスト + パルスアニメ(グリーン)
- オープナー: `#ff4b4b`背景 + 白テキスト
- 未行動: ライトグレー背景

### トランプカード
- 白背景 + 角丸`12px`
- ソフトシャドウ: `0 4px 12px rgba(0,0,0,0.1)`
- 内側ボーダー: 薄いグレー(`#e0e0e0`)
- スート色: ハート/ダイヤ=`#ff4b4b`, スペード/クラブ=`#3c3c3c`

### スタッツ表示
- 白カード背景 + ソフトシャドウ
- 正解率リング: `#58cc02`(グリーン)
- ストリーク: `#ff9600`(オレンジ)のアクセント

### タブバー
- 白背景 + 上部に薄いボーダー(`#e0e0e0`)
- 非アクティブ: `#afafaf`
- アクティブ: `#58cc02` + 上部ドットインジケーター

### フィードバック
- 正解: 画面が緑にフラッシュ + 大きな丸テキスト「ナイス！」
- 不正解: 赤フラッシュ + 震え(shake)エフェクト
- 紙吹雪: 正解時に小さなパーティクル

### チップ
- シンプルな円形に変更(リアルなテクスチャ廃止)
- ブラインド: `#78909c` (ブルーグレー)
- レイズ: `#ff4b4b`
- ヒーロー: `#ff9600`

### Dボタン
- 白背景 + `#58cc02`ボーダー + 「D」テキスト

## body・背景
- `background: #f7f7f7`
- ノイズテクスチャ・アンビエントグロー廃止
- クリーンなフラット背景

## アニメーション
- カードディール: 継続(タイミング微調整)
- ボタンタップ: border-bottomが縮んで「押し込まれる」効果
- 正解/不正解フラッシュ: 継続
- ヒーローパルス: グリーン系に変更

## レスポンシブ
- スマホファースト(max-width: 480px がメイン)
- 既存の360pxブレークポイント継続
